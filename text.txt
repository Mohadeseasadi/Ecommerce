🎯 E-commerce Backend with NestJS & PostgreSQL

طراحی و پیاده‌سازی یک سیستم فروشگاه آنلاین با استفاده از NestJS برای معماری مدرن و مقیاس‌پذیر و PostgreSQL برای ذخیره‌سازی داده‌های رابطه‌ای.

📦 ویژگی‌ها:

احراز هویت JWT و نقش‌بندی کاربران (کاربر، مدیر)

مدیریت محصولات، دسته‌بندی‌ها، کاربران و سفارش‌ها

ایجاد سفارش با محاسبه قیمت نهایی و اتصال به سبد خرید

ذخیره داده‌ها با TypeORM و ارتباطات OneToMany و ManyToOne

استفاده از Docker برای راه‌اندازی PostgreSQL در محیط توسعه

مستندسازی API با Swagger

مدیریت خطاها و اعتبارسنجی درخواست‌ها با Pipes و Exception Filters

🚀 هدف از این پروژه:
تمرین توسعه ساختار‌یافته با NestJS و پیاده‌سازی اصول Domain-driven Design در کنار یک پایگاه داده SQL.
